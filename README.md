# NeoVim
* **Navchad/custom**: 
  * **What**: For Navchad preconfiguration custom 
  * **How to use**: 
    1. Refers to NvChad Plugin Document and install it
    2. Put "custom" folder directly to ~/.config/nvim/lua/[here]
    3. Run ```:PackerSync``` in nvim
